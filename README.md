# Wellcraft Boat Manager

ניהול כלכלי־תפעולי לסירת וולקראפט:
- יומן יציאות (אורחים, תשלומים, דגים)
- הוצאות (דלק, עגינה, טיפולים, ציוד)
- תזרים שותפים (כמה כל שותף הכניס והוציא)
- סיכומים חודשיים לסירה

הפרויקט כרגע עובד על מודל נתונים (סכמות + אקסלים/CSV),
ובהמשך יכול להתפתח לסקריפטים, API או דשבורד.
